## Methodology

* Used

	* HTML elements
	* CSS selectors
	* CSS style attributes
	* Fixed nav-bar with anchor scroll offsets
	* Box model
	* CSS Media Queries
	* Flex 
	* External CSS files


* Not used

	* Grid layouts
	* Bootstrap
	* JavaScript
		* and so on...
